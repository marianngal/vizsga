11. megcsináljuk az adat interface-t (az osztályon kívül!):
interface KekturaAdat {
  start: string;
  end: string;
  distance: number;
  ascent: number;
  descent: number;
  type: string;
}

12. KekturaElem osztály elkészítése a Kektura Adat interface-ből:
class KekturaElem implements KekturaAdat {
  start: string;
  end: string;
  distance: number;
  ascent: number;
  descent: number;
  type: string;

  constructor(adat: string) {
    const reszletek = adat.split(";");
    this.start = reszletek[0];
    this.end = reszletek[1];
    this.distance = parseFloat(reszletek[2].replace(",", "."));
    this.ascent = parseInt(reszletek[3]);
    this.descent = parseInt(reszletek[4]);
    this.type = reszletek[5];
  }

13. KékturaBetöltő függvény készítése az osztályon belül!, az interface fölé:
Kekturafeltolto(feltoltendoElem: string[]): KekturaElem[] {
  let beolvasottAdatok: KekturaElem[] = [];
  for (let i = 0; i < feltoltendoElem.length; i++) {
    beolvasottAdatok.push(new KekturaElem(feltoltendoElem[i]));
  }
  return beolvasottAdatok;
 }

14. a feltöltő függvény használata az osztályon belül!, azaz az objektum-tömb létrehozása a forrás-adatokból:

 kekturaAdatok: KekturaElem[] = this.Kekturafeltolto(this.adatok);

} <-- itt van az osztály vége!
